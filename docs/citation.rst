Citation
--------

If you use OpenPyTEA in academic work, please cite it. See the
`CITATION.cff <https://github.com/PBTamarona/OpenPyTEA/blob/main/CITATION.cff>`_
file for the full citation metadata, or use:

.. code-block:: bibtex

   @software{tamarona_openpytea,
     author  = {Tamarona, Panji B. and Vlugt, Thijs J.H. and Ramdin, Mahinder},
     title   = {{OpenPyTEA}: An open-source Python toolkit for techno-economic
                assessment of process plants},
     version = {2.1.0},
     url     = {https://github.com/PBTamarona/OpenPyTEA},
   }
